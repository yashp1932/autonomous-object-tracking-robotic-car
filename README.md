# Autonomous Object-Tracking Robotic Car 

## 锔 **Summary**  

A dual-mode robotic car built on an **Raspberry Pi 4** that can autonomously track objects or be manually driven via a **PS4 controller**. Implemented in **C++** using **OpenCV** for computer vision, **pigpio** for low-level PWM motor/servo control, and the **Linux joystick API** for input. Optimized for low-latency control (<50 ms) and reliable tracking accuracy (卤3 px).  

**Tech Stack:** C++ 路 OpenCV 路 pigpio 路 GStreamer 路 Linux Joystick API 路 Raspberry Pi 4 路 TB6612FNG Motor Driver 路 Servo + DC Motors  

**Key Features:** Autonomous HSV-based object tracking 路 manual override via PS4 controller 路 low-latency camera + control loop 路 custom mechanical design
